Method

Summary

  • Input: Vectorized HD map (lane graph + geometry) and desired number/types of agents.
  • Map Encoder: Relation-aware message-passing attention over lane topology produces compact map latents.
  • Agent Encoder: IAGE-biased self-attention captures timestep-unconditioned interaction priors, yielding agent latents.
  • Diffusion Denoiser: AdaLN-conditioned denoiser iteratively refines noisy agent states via IAGE self-attention, AMGE agent–map cross-attention, and a pointwise FFN.
  • Output: Diverse, physically valid and map-consistent initial scene configurations for downstream simulation.

Key components

IAGE
Interaction-Aware Geometry Encoder — biases agent self-attention with pairwise geometric relations.
AMGE
Agent–Map Geometry Encoder — biases cross-attention between agents and map latents.
AdaLN
Adaptive Layer Normalization — injects diffusion timestep conditioning via scale-and-shift.
Losses
Diffusion objective + unified safety, alignment, and reconstruction auxiliary losses.
